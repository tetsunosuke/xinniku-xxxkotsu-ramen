# 〇ンニク〇んこつラーメン (プロジェクト名: 澱みのスープ)

本作は、プレイヤー自身のスマートフォン画面と、道端で「拾ったスマートフォン」の画面を行き来しながら謎を解き明かす、仮想OS型のアドベンチャー（ARG）ホラーゲームです。

## 📖 ゲームの概要

プレイヤーは、拾ったスマートフォンの中に残された手掛かり（SNSの投稿、写真フォルダ、ブラウザの閲覧履歴、メッセージアプリなど）から、持ち主である失踪した動画配信者「タベアルキ太郎」の行方を追います。
日常に潜む狂気と、徐々に明らかになる「〇ンニク〇んこつラーメン」の恐るべき真実がプレイヤーを待ち受けています。

## ✨ 主な特徴

- **デュアルデバイス・システム**: 自分のスマホ画面（友人とのやり取り）と、拾ったスマホ画面（事件の調査）をワンタップで切り替えながら進行。
- **SNSを利用したパスコード推測**: 持ち主の公開タイムラインや知人の投稿を紐解き、誕生日や記念日からパスコードを特定するリアルなハッキング体験。
- **インタラクティブな証拠集め**: 写真のズームによるアノテーションの発見や、ブラウザでのニュース記事の「隠しリンク」クリックによる画面の変貌ギミック（文字化け・ホラー演出）。
- **マルチエンディング**: 最終局面における3つの質問への回答によって、救出成功（グッドエンド）か、自らも闇に飲まれるか（バッドエンド）が分岐します。

## 🛠 使用技術

- **フレームワーク**: Astro (開発サーバーおよびルーティングの基盤として使用)
- **フロントエンド**: HTML5, Vanilla JavaScript, CSS3
- **通信設計**: `window.postMessage` を利用した iframe 内アプリと親OS間の非同期メッセージング

## 📂 ディレクトリ構成

```text
/
├── public/                 # 静的ファイル群（このディレクトリの中身がルートとして配信される）
│   ├── index.html          # ゲームのメインOS（ルートUI）
│   ├── apps/               # iframeで読み込まれる各仮想アプリ
│   │   ├── browser.html    # ブラウザアプリ（ニュース、ラーメン店HP）
│   │   ├── chat.html       # LIMEアプリ（ケンとのチャット、クイズ）
│   │   ├── gallery.html    # 写真アプリ（証拠探し）
│   │   └── sns.html        # SNSアプリ（パスコード推測、ストーリー背景）
│   └── assets/             # スタイルシート、画像、コアスクリプト
│       ├── css/
│       ├── images/
│       └── js/
│           └── os-core.js  # OSのステータス管理やアプリ起動を司るコアロジック
├── src/                    # Astroのソースディレクトリ
│   └── pages/
│       ├── index.astro     # public/index.html を読み込んで配信するエントリーポイント
│       └── hint.astro      # ゲームの完全攻略・ヒントページ
├── package.json            # 依存関係およびnpmスクリプト
└── astro.config.mjs        # Astroの設定ファイル
```

## 🚀 起動方法

依存関係をインストールし、開発サーバーを起動してください。

```bash
npm install
npm run dev
```

起動後、ブラウザで `http://localhost:4321/` にアクセスするとゲームをプレイできます。
攻略情報やヒントを確認したい場合は、`http://localhost:4321/hint` にアクセスしてください。

---

*※キャッシュが残って表示が更新されない場合は、ブラウザの「サイトデータ消去」を行うか、`npm run dev:force` または `npm run clean` をお試しください。*
